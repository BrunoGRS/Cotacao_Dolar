==========================================

Aplicação de Cotação do dólar

==========================================

Funcionalidade: Faz a requisição para uma API de cotação de moedas e retornar um 
dicionário com dados dessa cotação, o dicionário é tratado e mostrado o valor da cotação
no momento autal.

Possui interface grática feita através do PySimpleGUI.
